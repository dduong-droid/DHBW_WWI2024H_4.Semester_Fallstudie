# Infra

Dieser Ordner ist fuer Entwickler 3 vorgesehen.

Hier hinein gehoeren spaeter unter anderem:

- Datenbank- und Migrationsdateien
- Auth- und Session-nahe Infrastruktur
- lokale Dev-Konfiguration fuer Persistenz
- Supabase- oder vergleichbare Setup-Dateien
- ENV-Dokumentation und technische Anschlussstellen

Aktuell ist der Bereich absichtlich noch leichtgewichtig, damit die Infra sauber entlang der bestehenden Backend-Domainen aufgebaut werden kann.
